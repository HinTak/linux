Device-Tree bindings for SUNXI IR controller found in sunXi SoC family

Required properties:
- compatible	    : "nvt,nvt-irs"
- interrupts	    : should contain IRS IRQ number;
- reg		    : should contain IO map address for IRS.

Optional properties:

Example:

nvt_irs: nvt_irs@fd0ac000 {
    compatible = "nvt,nvt-irs";
    reg = <0xfd0ac000 0x200>;
    interrupts = <0x0 0xb2 0x4>;
};
